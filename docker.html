1. Giới thiệu về Docker

1.1 Docker là gì?

Docker là một nền tảng mở để phát triển, vận chuyển và chạy ứng dụng. Docker cho phép tách ứng dụng khỏi cơ sở hạ tầng, giúp phần mềm chạy đồng nhất trên mọi môi trường.

1.2 Lợi ích của Docker

  • Nhẹ và nhanh hơn máy ảo
  • Tính di động cao
  • Dễ dàng mở rộng
  • Tính cách ly
  • Tái sử dụng và chia sẻ

2. Kiến trúc Docker

2.1 Các thành phần chính

- Docker Engine
    ├── Docker Daemon (dockerd)
    ├── REST API
    └── Docker CLI (docker)

- Docker Objects
    ├── Images
    ├── Containers
    ├── Networks
    └── Volumes

2.2 Docker Engine

Docker Engine bao gồm:

  • Docker Daemon: Quản lý các objects của Docker
  • REST API: Giao tiếp với Docker Daemon
  • Docker CLI: Giao diện dòng lệnh của Docker

3. Cài đặt Docker

3.1 Cài đặt trên Ubuntu

# Cập nhật package index
sudo apt-get update

# Cài đặt các gói cần thiết
sudo apt-get install apt-transport-https ca-certificates curl gnupg lsb-release

# Thêm Docker GPG key
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

# Thiết lập repository
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Cài đặt Docker Engine
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

3.2 Kiểm tra cài đặt

# Kiểm tra phiên bản
docker --version

# Chạy container test
docker run hello-world

4. Docker Images

4.1 Quản lý Images

# Liệt kê images
docker images

# Tải image
docker pull image_name:tag

# Xoá image
docker image rm image_name
docker rmi image_name

# Tìm kiếm image
docker search image_name

# Lưu image ra file
docker save -o image.tar image_name

# Nạp image từ file
docker load -i image.tar

4.2 Tạo Image

# Từ Dockerfile
docker build -t image_name:tag .

# Từ container
docker commit container_name new_image_name
