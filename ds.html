Hiểu về Các Cấu Trúc Dữ Liệu trong Dự Án Node.js

Trang này giải thích các cấu trúc dữ liệu phổ biến và cách chúng có thể được áp dụng trong các dự án thực tế sử dụng Node.js, Express, và TypeScript với Generics.

1. Mảng (Array)

Mảng là một cấu trúc dữ liệu lưu trữ một danh sách các phần tử theo thứ tự nhất định. Mảng thường được sử dụng khi bạn cần lưu trữ nhiều mục giống nhau. 

const users: Array<{ id: number; name: string }> = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' }
];
console.log(users.map(user => user.name)); // ['Alice', 'Bob']

2. Đối Tượng (Object)

Đối tượng trong JavaScript và TypeScript được sử dụng để lưu trữ dữ liệu theo dạng các cặp khóa-giá trị. Nó là một trong những cấu trúc dữ liệu cơ bản và phổ biến nhất. 

interface User {
  id: number;
  name: string;
  age: number;
}

const user: User = {
  id: 1,
  name: 'Alice',
  age: 28
};

console.log(user.name); // "Alice"

3. Hash Table (Bảng Băm)

Bảng băm là một cấu trúc dữ liệu ánh xạ giữa khóa và giá trị. Với bảng băm, bạn có thể truy cập, thêm, hoặc xóa các phần tử rất nhanh chóng. 

class HashTable {
  private table: Map;

  constructor() {
    this.table = new Map();
  }

  insert(key: K, value: V): void {
    this.table.set(key, value);
  }

  search(key: K): V | undefined {
    return this.table.get(key);
  }
}

const hashTable = new HashTable();
hashTable.insert('user1', 'Alice');
console.log(hashTable.search('user1')); // "Alice"

4. Ngăn Xếp (Stack)

Ngăn xếp (Stack) là cấu trúc dữ liệu kiểu LIFO (Last In, First Out), tức là phần tử cuối cùng được thêm vào sẽ là phần tử đầu tiên bị lấy ra. 

class Stack {
  private items: T[] = [];

  push(element: T): void {
    this.items.push(element);
  }

  pop(): T | undefined {
    return this.items.pop();
  }
}

const stack = new Stack();
stack.push(10);
stack.push(20);
console.log(stack.pop()); // 20

5. Hàng Đợi (Queue)

Hàng đợi (Queue) là cấu trúc dữ liệu kiểu FIFO (First In, First Out), phần tử đầu tiên được thêm vào sẽ là phần tử đầu tiên bị lấy ra. 

class Queue {
  private items: T[] = [];

  enqueue(element: T): void {
    this.items.push(element);
  }

  dequeue(): T | undefined {
    return this.items.shift();
  }
}

const queue = new Queue();
queue.enqueue('task1');
queue.enqueue('task2');
console.log(queue.dequeue()); // "task1"

6. Set (Tập hợp)

Set là một cấu trúc dữ liệu lưu trữ các phần tử duy nhất, không có phần tử trùng lặp. Nó có thể được sử dụng để lưu trữ các giá trị mà không cần quan tâm đến thứ tự. 

const numbers = new Set();
numbers.add(1);
numbers.add(2);
numbers.add(2); // Phần tử trùng sẽ không được thêm vào
console.log(numbers); // Set { 1, 2 }

const strings = new Set();
strings.add("Alice");
strings.add("Bob");
strings.add("Alice"); // Phần tử "Alice" đã có, không thêm lại
console.log(strings); // Set { 'Alice', 'Bob' }

7. Danh Sách Liên Kết (Linked List)

Danh sách liên kết là một cấu trúc dữ liệu trong đó mỗi phần tử lưu trữ dữ liệu và một con trỏ hoặc tham chiếu đến phần tử tiếp theo. 

class Node {
  data: T;
  next: Node | null = null;

  constructor(data: T) {
    this.data = data;
  }
}

class LinkedList {
  head: Node | null = null;

  insert(data: T): void {
    const newNode = new Node(data);
    newNode.next = this.head;
    this.head = newNode;
  }
}

const list = new LinkedList();
list.insert(1);
list.insert(2);
console.log(list.head); // { data: 2, next: { data: 1, next: null } }

8. Cây (Tree)

Cây là một cấu trúc dữ liệu có nút gốc và các nút con, thường được sử dụng để lưu trữ dữ liệu có cấu trúc phân cấp như thư mục hoặc hệ thống tệp. 

class TreeNode {
  value: T;
  left: TreeNode | null = null;
  right: TreeNode | null = null;

  constructor(value: T) {
    this.value = value;
  }
}

class BinaryTree {
  root: TreeNode | null = null;

  insert(value: T): void {
    const newNode = new TreeNode(value);
    if (!this.root) {
      this.root = newNode;
    } else {
      this.insertNode(this.root, newNode);
    }
  }

  insertNode(node: TreeNode, newNode: TreeNode): void {
    if (newNode.value < node.value) {
      if (node.left === null) {
        node.left = newNode;
      } else {
        this.insertNode(node.left, newNode);
      }
    } else {
      if (node.right === null) {
        node.right = newNode;
      } else {
        this.insertNode(node.right, newNode);
      }
    }
  }
}

const tree = new BinaryTree();
tree.insert(10);
tree.insert(5);
tree.insert(15);
console.log(tree.root); // Root node with value 10

9. Đồ Thị (Graph)

Đồ thị là một cấu trúc dữ liệu gồm các đỉnh (nodes) và các cạnh (edges) nối các đỉnh lại với nhau. Đồ thị có thể có chu trình hoặc không. 

class Graph {
  private adjList: Map = new Map();

  addVertex(vertex: string): void {
    if (!this.adjList.has(vertex)) {
      this.adjList.set(vertex, []);
    }
  }

  addEdge(vertex1: string, vertex2: string): void {
    this.adjList.get(vertex1)?.push(vertex2);
    this.adjList.get(vertex2)?.push(vertex1);
  }

  getAdjList(): Map {
    return this.adjList;
  }
}

const graph = new Graph();
graph.addVertex("A");
graph.addVertex("B");
graph.addEdge("A", "B");
console.log(graph.getAdjList()); // Map { 'A' => ['B'], 'B' => ['A'] }

10. Heaps

Heap là một cây nhị phân đặc biệt trong đó phần tử cha luôn có giá trị lớn hơn hoặc nhỏ hơn các phần tử con. 

class MaxHeap {
  private heap: number[] = [];

  insert(value: number): void {
    this.heap.push(value);
    this.bubbleUp();
  }

  private bubbleUp(): void {
    let index = this.heap.length - 1;
    while (index > 0) {
      const parentIndex = Math.floor((index - 1) / 2);
      if (this.heap[index] <= this.heap[parentIndex]) break;
      [this.heap[index], this.heap[parentIndex]] = [this.heap[parentIndex], this.heap[index]];
      index = parentIndex;
    }
  }

  getHeap(): number[] {
    return this.heap;
  }
}

const heap = new MaxHeap();
heap.insert(10);
heap.insert(20);
console.log(heap.getHeap()); // [20, 10]
